
# Learning Journey

## User Story

As a teacher,
I want to see everything about my learner in one place,
so I can prepare, teach, and evaluate without searching through multiple screens.

---

## UX Goal

The teacher should understand the learner's current situation in less than 10 seconds.

---

## Primary Actions

- View progress
- Start session
- Add note
- Add assessment
- Create goal
- Schedule session
