
# Actors

## Mentor

المدرس أو المدرب الذي يستخدم Numo لإدارة عمله اليومي.

المهام:
- إدارة الطلاب.
- إنشاء الجلسات.
- تسجيل الحضور.
- متابعة التقدم.
- الاطلاع على التقارير.

---

## Learner

المتعلم الذي يحضر الجلسات ويتابع رحلته التعليمية.

المهام:
- رؤية الجلسات.
- متابعة التقدم.
- رؤية الأهداف.
- استلام الإشعارات.

---

## Admin

يدير النظام بالكامل.

المهام:
- إدارة المستخدمين.
- إدارة المنصة.
- إدارة الصلاحيات.

---

## Organization (Future)

مدرسة أو معهد يستخدم Numo لإدارة عدة مدرسين.
